CSS Selectors

📖 Selector Guide & Examples

Supported Selectors:

  • Title - Product name/heading
  • Price - Current price (numeric value)
  • Image - Product image URL
  • Availability - Stock status, sale badges, etc.
  • SKU - Product identifier/code
  • Brand - Manufacturer/brand name
  • Description - Product description text

Example Selectors:

  • h1.product-title - Title by class
  • .price-current span - Price in nested element
  • img[data-src] - Image with data attribute
  • #availability-status - Availability by ID

💡 Tips:

  • Add multiple selectors as fallbacks (tried in order)
  • Use browser DevTools to inspect elements
  • Test selectors on the actual product page first
